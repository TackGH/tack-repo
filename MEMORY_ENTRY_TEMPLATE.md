MEMORY.md entry template

- Title: Short descriptive title
- Date: YYYY-MM-DD
- Summary: One-line summary
- Tags: comma-separated (#project:Tack, #preference, #decision)
- Source: memory/YYYY-MM-DD.md#Lnn (line reference)
- Hash: sha1/content-hash

Example:
- Title: Default model preference
- Date: 2026-02-27
- Summary: Default chat model set to openai/gpt-5-mini; tool runs use openai/gpt-5.1-codex.
- Tags: #preference, #model
- Source: memory/2026-02-27.md#L1
- Hash: 9f2a...
