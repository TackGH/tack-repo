Enterprise features (collected):
- TackCloud: unified enterprise dashboard for device management, live maps, device health, sensor streaming, alerts & automations.
- Sentinel: enterprise GPS tracking for fleets, events, and assets; hardware integration for wired sensors, scale to thousands of units; API integration mentioned for fleet management and dashboard access.
- EnAware X: solar-powered RTU/sensor hub; supports Modbus, I2C, Analog, Pulse; IP67, GPS, 4G connectivity; remote configuration and local triggers.
- FloodFinder: AI-driven flood detection sensors (B200, U300/U1500, R5000) with push/email/SMS alerts, GPS geo-tagging, solar power, compact form factor — aimed at urban flood resilience.
